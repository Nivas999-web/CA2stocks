## Setup

1. Clone the repo or unzip
2. Install dependencies

```bash
npm install
```

3. Create a `.env` file in root:

```
VITE_OPENAI_API_KEY=your_openai_key_here
VITE_FINNHUB_API_KEY=your_finnhub_key_here
```

4. Start the dev server:

```bash
npm run dev
```
